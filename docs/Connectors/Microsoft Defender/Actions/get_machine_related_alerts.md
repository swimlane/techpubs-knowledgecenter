# Get Machine Related Alerts

**Description**: Retrieve alerts from Microsoft Defender related to a specific machine by providing the machine's unique ID.

## Endpoint

- **URL:** `/api/machines/{{id}}/alerts`
- **Method:** `GET`
## Inputs

- **path_parameters** (object) – Required
  - **id** (string) – Required
