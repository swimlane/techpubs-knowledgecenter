Getting Started Guide

This guide contains scenarios and step-by-step instructions that walk you through how to use Swimlane. Use the topics here to learn the basics about Swimlane applications, workflow, tasks, and automations. We hope this helps you begin to realize the many possibilities of what you can do with Swimlane.

Important! Ensure that you have a fully-installed deployment of the Swimlane platform.

Also Important! You must be a Swimlane administrator in order to complete all of the scenarios included in this guide.
