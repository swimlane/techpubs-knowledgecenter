Test the Latest Automation in Records

To test the latest automations in records:

  1. From the global navigation, hover over Application Records and select + to add a new record.

  2. On New Record, enter your name in Text (2).

  3. Tab out of the Text (2) field, then click Integration.

    You will be prompted to save the record. Click OK.

  4. Swimlane saves your record, and then runs the integration task. You'll receive notification when the integration task has finished.

  5. The integration task has changed the Description field text. Take a look!
