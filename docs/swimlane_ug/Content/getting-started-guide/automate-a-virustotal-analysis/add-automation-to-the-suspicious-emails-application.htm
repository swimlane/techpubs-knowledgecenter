Add Automation to the Suspicious Emails Application

Next you will add two forms of automation to your Suspicious Emails application: a manual push button, and a workflow action.

To add automation to the Suspicious Emails application:

  1. From the global navigation menu, select Applications and Applets, and then locate and open your Suspicious Emails application.

  2. In Application Builder, select Integration from within the Layout Objects. Drag and Drop it to the Email Import section of the Application Form Layout.
