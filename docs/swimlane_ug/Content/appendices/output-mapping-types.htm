Mapping Types

This appendix outlines the allowable output fields by field data type.

Note: If an application’s field doesn’t currently conform to the allowable output mapping rules, you will need to create and map to a new application field that does conform.

Also Note: If a plugin's output type changes in a new version or update of the plugin, you will need to remap outputs.
