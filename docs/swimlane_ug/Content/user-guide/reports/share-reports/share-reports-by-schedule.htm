Share Reports by Schedule

On the report page, access the ellipsis drop-down menu and select Schedule to access the Schedule Report page.

This feature allows you to schedule when an email of the report is sent to recipients. In addition to scheduling the email, you can also choose to send the email now and choose to download a preview of the report.

To schedule a report:

  1. On Schedule Report, go to Schedule Settings. Select either Recurring or One Time for the frequency of reports being sent. For Recurring, select how often you would like the report to send. This can be scheduled daily, weekly, or monthly at a specified time. For One Time, select the day and time you would like the schedule to send.

  2. Toggle Schedule Disabled to enable the schedule. If the schedule is not enabled, it will not send.

  3. Under Report Details, under Name, type a name for the report. This name is visible in the Report Preview once you click out of the Name text box.

    4. The name is the subject line of the email.

  1. Under Notes, type notes to include in the report. These notes are visible in the Report Preview once you click out of the Notes text box.

    2. Notes display as the body of the email.

  1. Under Recipients, search for and select the name of the Swimlane users or groups you would like to add or enter an email address and then select Add Value.

    2. The recipients that display in this field are other Swimlane users that have role-based access to the report. Select from those usernames. Know that you can also manually enter email address for people outside of Swimlane users.

    Note: Users who have previously been added to scheduled reports will continue to have access. You should edit the schedule to include them; use their email addresses instead of their username and the scheduled report will continue to be delivered.

    Note: The Name and Recipients fields are required to send your report.

  1. Under Report Controls, you can chose to include a link to the report in Swimlane, to include the report table as a CSV file attachment, to include a report table, and select the page orientation of the report.

    2. Note: When including a report table, the table displays a maximum of 25 records. If the report contains more than 25 records, you receive a message at the bottom of the report table that indicates how many total records are included.

    Note:Unavailable Report Controls appearsif there is a report containing a widget, or if the report does not contain a chart. This lets you know that the report will be including the report table since widgets are not supported in shared reports and the chart cannot be displayed.

  1. Click the drop-down arrow next to Header and Footer Formatting. This is where you format the header and footer for your report.

    2. Note: The header and footer can be customized using the available editor. To make more advanced changes to the layout, you can access and edit the raw HTML of the header and footer by opening the source code tool from the editor's toolbar.

  1. Once you have finished editing your report, click Save Schedule. A dialog appears confirming the email has been scheduled. Once the schedule is saved, click Close to exit.

    2. To view all scheduled reports, click the ellipsis on the report header and select Details and Schedules.

    Your email shows the name of the account the email is sent from, the name of the report, the notes included in the report, a link to the report in Swimlane if selected, a PDF of the report, and a CSV file of the report table if selected.

    Note: You can configure the mail and PDF default settings in Settings under Email and PDF Settings. Changes made to the PDF default settings will show in scheduled reports that have not had the header or footer edited. Scheduled reports that have been edited will not reflect these changes in the PDF default settings.
