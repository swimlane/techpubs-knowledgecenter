Download Reports

On the report page, access the ellipsis drop-down menu and select Download to access the Download Report page.

  1. Once in Download Report, under Name, type a name for your report. This name is visible in the Report Preview once you click out of the Name text box.

Note: A report name is required to download your report.

  1. Under Notes, type any notes you would like to include on the report. These notes are visible in the Report Preview once you click out of the Notes text box.

  2. Under Report Controls, you can chose to include the report table as a CSV file attachment, to include a report table, and select the page orientation of the report.

    3. Note: When including a report table, the table displays a maximum of 25 records. If the report contains more than 25 records, you receive a message at the bottom of the report table that indicates how many total records are included.

    Note:Unavailable Report Controls appears if there is a report containing a widget, or if the report does not contain a chart. This lets you know that the report will be including the report table since widgets are not supported in shared reports and the chart cannot be displayed.

  1. Click the drop-down arrow next to Header and Footer Formatting. This is where you format the header and footer for your report.

    2. Note: The header and footer can be customized using the available editor. To make more advanced changes to the layout, you can access and edit the raw HTML of the header and footer by opening the source code tool from the editor's toolbar.

  1. Once you have finished editing your report, click Download. A green check appears on the Download button once your report is downloaded. Click Close to exit.
