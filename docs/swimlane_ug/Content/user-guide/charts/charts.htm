Charts

You can visualize data and create reports with charts.

To view charts:

  1. Open the Default Report (or any report) view of Application Records.

  2. Click Charts on the Default Report taskbar.

    3. The visualization panel is split into two parts: chart configuration on the left and preview on the right.

    To create a visualization, select the desired chart type. Chart types are listed above the chart preview, and are grouped into categories.
