Extend Third-Party Certificates for Python Package Installation

This topic outlines how to utilize third-party certificates for the Python package installation utility in the UI.

Important! Before you begin you must set up your certificates as indicated in the Third-Party Certificates Volume topic of this guide.

To extend third-party certificates for Python package installation:

  1. Modify the files at /opt/swimlane/.secrets/.api-env & /opt/swimlane/.secrets/.tasks-env to include:

    2. PIP_CERT=/etc/ssl/certs/ca-certificates.crt

  1. Restart containers.

  2. In a CLI, enter:

    3. cd /opt/swimlane

    docker-compose down

    docker-compose up -d
