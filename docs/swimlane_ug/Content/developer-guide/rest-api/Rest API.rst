Rest API
========

Swimlane exposes a REST API that allows you to work with Swimlane data
programmatically.

Click to download a PDF Reference of the Swimlane API:

`Swimlane API Reference Document
(PDF) <https://docs.swimlane.com/docs/resources/images/files/swimlaneapiv2.pdf>`__

To access this from within Swimlane, as an administrator, click
**Settings** and then click **API Docs**. A browser tab opens where you
can interact with the Swimlane API.
