Edit Reports

Reports are collections of records collected from a Swimlane application. Reports can be customized through keyword searching and filtering, saved to show the selected information, and shared with other users.

You can create a new report from either APPLICATION RECORDS or REPORTS. For details on how to create a new report, go to Create and Save Reports.

Editing a Report

To edit a report, from the global navigation menu select REPORTS. Locate your report, and then click the pencil icon.

Alternatively, to edit the current report, click the ellipsis on the report taskbar and open the menu. Click Details and Schedules.

This opens Report Details and Schedules. From here you can edit the settings for the report, send scheduled reports, or create new schedules.

Deleting a Report

To delete a report:

  1. On the Reports page, select and load the report that you want to delete.

  2. Click the ellipsis on the Report taskbar and open the menu, then select Delete Report.

    3. After you have selected to delete the report, a dialog box appears asking you to confirm that you want to delete the report.

  1. Click Delete to delete the report.

    2. A dialog box appears to show you that your dashboard has been successfully deleted.
