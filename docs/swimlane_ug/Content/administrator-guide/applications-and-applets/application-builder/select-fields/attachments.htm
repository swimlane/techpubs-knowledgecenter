Attachments

Use this field to store files in a record. The list of preview options are: .csv, .doc, .docx, .gif, .jpeg, .jpg, .pdf, .png, .ppt, .pptx, .tsv, .txt, .xls, and .xlsx. There are no document-type limitations for attachments.

To create an attachments field:

From Application Builder's Field Types, select Attachments and then drag and drop it to the Form Layout. Drop the field in the layout area, or within a tab or section layout object.

Access the field's Field Properties tab and complete the following fields as needed:

Field Step Example
Display Name Enter the name of the field. Related Files
Help Text Enter contextual help text. You will first need to specify whether the help text will appear above or below the field in the record form, and then you can enter the text. Attach related files here.
Read-only Click to indicate that the field is read-only for the record. The field will not be editable. checkmark

Next, consider the following advanced options in the Field Properties tab:

Field Step Example
Max Size Use to specify the maximum allowed file size for the attachment. 100,000 kb
Delete Attachments Select this field to indicate that the attachment be deleted. select
Time to live (days) Use to indicate the number of days before the attachment is deleted. 30 (days)

Add specific field-level permissions by role, if needed, and then click Apply.
