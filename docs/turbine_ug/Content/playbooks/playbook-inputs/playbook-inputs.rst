Playbook Inputs
===============

Playbook inputs pass in and normalize data that you can use as inputs
for playbook actions, including connector actions and nested playbooks.

This section covers features including:

-  Create and Edit Playbooks

-  Remove Playbook Inputs

-  Promote and Map Playbook Inputs to Outputs
