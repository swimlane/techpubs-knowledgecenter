
Actions
=======
Available actions for Searchlight Connector

+------------------------------------------------------+------------------------------------------------------+
| Action                                               | Description                                          |
+======================================================+======================================================+
| Set Triage Item State                                | Set the state for a triage item.                     |
+------------------------------------------------------+------------------------------------------------------+
| List Impersonating Domain Alerts                     | Retrieve a list of impersonating domain alerts.      |
+------------------------------------------------------+------------------------------------------------------+
| List Triage Items                                    | Retrieve triage items from given triage event IDs.   |
+------------------------------------------------------+------------------------------------------------------+
| List Alerts                                          | Retrieve a list of alerts.                           |
+------------------------------------------------------+------------------------------------------------------+
| Create Triage Item Comment                           | Create a new comment for a given triage item.        |
+------------------------------------------------------+------------------------------------------------------+
| List Incidents                                       | Retrieve a list of incidents.                        |
+------------------------------------------------------+------------------------------------------------------+
| List Triage Item Events                              | Retrieve a list of triage item events.               |
+------------------------------------------------------+------------------------------------------------------+
| List Triage Item Comments                            | Retrieve a list of comments for a given triage item. |
+------------------------------------------------------+------------------------------------------------------+
